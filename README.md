# gqlgen-gorm-example
This is a demo for practice graphql and via gorm to interactive with database

## Prerequisites
- [docker](https://www.docker.com/products/docker-desktop)

## Usage
start app, redis and mysql service
```bash
$ docker-compose up -d
```
